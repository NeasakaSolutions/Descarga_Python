# Instalación de la versión más reciente de Python en Windows
Este documento explica cómo descargar, instalar y verificar la última versión estable de Python en un sistema Windows.

### Descargar Python:

- Abrir el navegador.

- Ir a la página oficial de versiones:
```bash
https://www.python.org/downloads/
```

- Baja hasta “Looking for a specific release?”

- Haz clic en Python 3.10.x

- Descarga:
```bash
Windows installer (64-bit) ← recomendado
```

- Se debio de descargar el sigueinte archivo:
```bash
python-3.10.13-amd64.exe
```
